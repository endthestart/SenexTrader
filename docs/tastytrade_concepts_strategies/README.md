# TastyTrade Concepts & Strategies Reference

This folder contains educational reference material from TastyTrade covering options trading concepts, strategies, and market terminology.

## Purpose

These documents serve as a knowledge base for understanding:
- Options trading fundamentals
- Trading strategies and their mechanics
- Market terminology and concepts
- Greeks and options pricing
- Risk management principles

## Source

All content is sourced from https://www.tastylive.com/concepts-strategies/

## Organization

Files are organized alphabetically by concept/strategy name. Each file contains:
- Concept definition
- How it works
- Practical applications
- Related concepts

## File Count

70 reference documents

## Usage

Use these documents to:
- Understand TastyTrade terminology used in the platform
- Learn about strategies before implementing them
- Reference options concepts during development
- Ensure correct implementation of trading mechanics

## Note

This is reference documentation only. Implementation details specific to Senex Trader are in the main project codebase.
